## 📱 Mobile App Data Flow (Offline-First)

```
┌────────────────────────────────────────────────────────────────┐
│                        USER ACTION                             │
│              (Record Voice, Take Photo, etc.)                  │
└───────────────────────────┬────────────────────────────────────┘
                            │
                            ▼
┌────────────────────────────────────────────────────────────────┐
│                   LOCAL WATERMELONDB                           │
│                  (SQLite on Device)                            │
│                                                                │
│  Tables:                                                       │
│  • users, farms, visits                                        │
│  • voice_recordings, photos                                    │
│  • planting_events, product_batches                            │
│  • sync_queue                                                  │
│                                                                │
│  Status: ✅ SAVED IMMEDIATELY (works offline)                  │
└───────────────────────────┬────────────────────────────────────┘
                            │
                            ▼
                    ┌───────────────┐
                    │  SYNC QUEUE   │
                    │  (Pending)    │
                    └───────┬───────┘
                            │
                            │ Wait for internet...
                            │
                            ▼
                    ┌───────────────┐
                    │  NETWORK?     │
                    │  ┌──────┐     │
                    │  │ Yes  │     │
                    │  └──┬───┘     │
                    └─────┼─────────┘
                          │
                          ▼
┌────────────────────────────────────────────────────────────────┐
│                  SYNC SERVICE ACTIVATED                        │
│                                                                │
│  1. Pull Changes:  Backend → Local DB                         │
│  2. Push Changes:  Local DB → Backend                         │
│  3. Media Upload:  Files → MinIO                              │
│  4. Conflict Resolution: Last-write-wins                       │
│                                                                │
└───────────────────────────┬────────────────────────────────────┘
                            │
                            ▼
┌────────────────────────────────────────────────────────────────┐
│                 BACKEND PROCESSING                             │
│                                                                │
│  • Transcribe voice recordings (Whisper)                      │
│  • Identify plant species (PlantNet)                          │
│  • Calculate environmental metrics                            │
│  • Generate QR codes                                          │
│  • Update analytics                                           │
│                                                                │
└───────────────────────────┬────────────────────────────────────┘
                            │
                            ▼
┌────────────────────────────────────────────────────────────────┐
│              RESULTS SYNCED BACK TO DEVICE                     │
│                                                                │
│  Status: ✅ SYNCHRONIZED                                       │
└────────────────────────────────────────────────────────────────┘
```

## 🔄 Sync Protocol (WatermelonDB)

```
App Initialization
      │
      ▼
┌──────────────────┐
│  Check Network   │
└────────┬─────────┘
         │
    ┌────┴────┐
    │         │
OFFLINE   ONLINE
    │         │
    │         ▼
    │   ┌──────────────────┐
    │   │  Pull Changes    │
    │   │  from Backend    │
    │   │                  │
    │   │  GET /sync/pull  │
    │   │  ?since=timestamp│
    │   └────────┬─────────┘
    │            │
    │            ▼
    │   ┌──────────────────┐
    │   │ Apply Changes    │
    │   │ to Local DB      │
    │   │                  │
    │   │ • Insert new     │
    │   │ • Update changed │
    │   │ • Delete removed │
    │   └────────┬─────────┘
    │            │
    │            ▼
    │   ┌──────────────────┐
    │   │ Push Local       │
    │   │ Changes          │
    │   │                  │
    │   │ POST /sync/push  │
    │   │ {changes: [...]} │
    │   └────────┬─────────┘
    │            │
    │            ▼
    │   ┌──────────────────┐
    │   │ Media Upload     │
    │   │ (if needed)      │
    │   │                  │
    │   │ Photos, Audio    │
    │   └────────┬─────────┘
    │            │
    └────────────┼─────────────┐
                 │             │
                 ▼             ▼
         ┌──────────────────────┐
         │  App Ready to Use    │
         │  (All Data Current)  │
         └──────────────────────┘
```
