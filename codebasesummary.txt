## 🏗️ Architecture Highlights

### Offline-First Mobile App
```
User Action → Local WatermelonDB → Sync Queue → Backend API (when online)
```

### Voice Recording Pipeline
```
Audio Capture → Local Storage → Upload Queue → Whisper AI → Transcription → DB
```

### Plant Identification Flow
```
Photo Capture → Geotag → Upload Queue → PlantNet API → Species Data → DB
```

### QR Traceability Chain
```
Farm Data → Evaluation → Environmental Metrics → QR Generation → Consumer Scan
```

8 Tables Implemented:
1. **users** - User profiles with roles

5. **photos** - Geotagged images with species data
6. **planting_events** - Tree planting tracking
7. **product_batches** - QR-traceable products
8. **sync_queue** - Offline sync management
